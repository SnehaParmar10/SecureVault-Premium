# 🔒 SecureVault | Cryptographic Utility

A futuristic encryption/decryption interface designed for secure data handling and privacy concepts.

## ✨ Features
- **Data Encryption**: Secure Base64 + Key-salting logic for text obfuscation.
- **Key-Based Access**: Multi-layered secret key verification for decryption.
- **Cyber-Security Theme**: Dark purple neon aesthetic with glassmorphism alerts.
- **Interactive UI**: Real-time feedback and state management for encryption workflows.

## 🛠️ Tech Stack
- **Languages**: HTML5, CSS3, JavaScript.
- **Design Concepts**: Cryptography, Information Security, Symmetric Filtering.

## 🧠 Why I built this?
Security is a core pillar of Computer Science. I built SecureVault to showcase my commitment to building secure software and understanding how data is protected in transit and at rest.
